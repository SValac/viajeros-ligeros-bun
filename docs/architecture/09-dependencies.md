# 9. Dependencias Adicionales

## Validación (recomendado: Zod)

```bash
bun add zod
```

Alternativa más ligera con Valibot:

```bash
bun add valibot
```

---

## Manejo de fechas (opcional)

```bash
bun add date-fns
# o
bun add dayjs
```
